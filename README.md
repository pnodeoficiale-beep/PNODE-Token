## About

PNODE is the native utility token of the PocketNode app. Providers earn it by sharing their mobile device computing power with the decentralized network. All services within PocketNode can only be purchased with PNODE.

## Features

- Fixed supply (no mint, no burn)
- No owner, no admin functions
- Fully decentralized
- Gas-efficient custom errors
- ERC-20 / BEP-20 compatible
